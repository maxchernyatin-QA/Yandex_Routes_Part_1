В рамках проекта была поставлена цель протестировать валидацию полей ввода времени и адресов в веб-приложении "Яндекс. Маршруты".

Задачи:

1. Провести тест-анализ, выполнить декомпозицию логики полей ввода.

2. Выделить классы эквивалентности и граничные значения.

3. Составить тест-кейсы на валидацию полей ввода времени и адресов в форме, логику расчёта стоимости и времени поездки на собственном автомобиле.

4. Составить баг-репорты по результатам проведенного тестирования.

Выполнено:

1. Проведен тест-анализ, выделены основные блоки для тестирования логики полей ввода.

2. Выделены классы эквивалентности и граничные значения.

3. Составлено 70 тест-кейсов, выполнено тестирование, обнаружено 16 багов.

4. Составлено 16 баг-репортов.
