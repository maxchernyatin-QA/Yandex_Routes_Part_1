В рамках проекта поставлена цель потестировать веб-приложение "Яндекс. Маршруты.

Для реализации поставленной цели нужно было выполнить две задачи — протестировать:

1. Провести тест-анализ, выполнить декомпозицию логики полей формы.

2. Выделить классы эквивалентности и граничные значения.

3. Протестировать валидацию полей ввода времени и адресов в форме, логику расчёта стоимости и времени поездки на собственном автомобиле.

4. Составить баг-репорты по результатам проведенного тестирования.
